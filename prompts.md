# Prompts Used

## Strategy Selection
**Prompt:** "Analyze the hackathon problem statement for Harriot Inc. and identify the highest impact strategic area that combines data analysis with generative AI."
**Rationale:** Focused on "Experience Differentiation" because it directly impacts revenue and leverages both predictive (who to target) and generative (how to persuade) AI.

## Code Generation
**Prompt:** "Create a Streamlit app that visually demonstrates traveler segmentation and dynamic offer generation. Use a premium design aesthetic."
**Refinement:** "Ensure the 'GenAI' copy generation simulates different tonalities for Business vs. Leisure travelers."

## Model Design
**Prompt:** "Suggest a simple but effective ML pipeline for predicting booking probability using demographic and behavioral data."
**Outcome:** Selected K-Means for segmentation + Random Forest for conversion prediction as a robust MVP baseline.
